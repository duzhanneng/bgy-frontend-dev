# bgy-fronted-devp
===========================
## Getting Started
下载依赖

```bash
$ npm install or $ cnpm install
```

启动项目

```bash
$ npm start appName projectName
```

项目打包

```bash
$ npm run build
```

多语言转换(需全局安装excel-tobe-json2)

```bash
$ npm install excel-tobe-json2 -g
$ npm run excel
```

如果你所使用的过程中有任何疑问或者建议，欢迎提出！
