# bgy-fronted-devp
===========================
## Getting Started
Install dependencies.

```bash
$ npm install
```

Start server.

```bash
$ npm start
```

BuildDll.

```bash
$ npm run buildDll
```

buildAnalyse: build with BundleAnalyzerPlugin.

```bash
$ npm run buildAnalyse
```

build.

```bash
$ npm run build:[uat|pre|prd]
```
If success, app will be open in your default browser automatically.
